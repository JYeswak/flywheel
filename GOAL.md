# Flywheel Repo Goal

status: ready
repo: repo-local

Current repo-level goal: publish Flywheel as a portable agentic workflow
ecosystem that can be installed, inspected, tested, and operated from public
repo-local artifacts without relying on private machine state.

Acceptance:

- Public docs, install scripts, doctor/preflight checks, loop commands, and
  agent-lane compatibility tests pass from a clean public export.
- NTM and non-NTM workflows are documented as optional operating modes, not
  hidden prerequisites.
- Publication remains blocked until real public GitHub, release assets, hosted
  workflow runs, and final human signoff are verified.
