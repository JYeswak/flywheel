Local gates before hosted gates

GitHub Actions should be final approval, not the first place you find basic breakage.

Flywheel favors local containers, dry-runs, doctor receipts, and screenshot checks before spending hosted CI minutes. That makes the public path cheaper, faster, and easier to trust.
