Local preflight

GitHub Actions should be final approval, not the first place you find basic breakage.

Flywheel favors local containers, dry-runs, doctor receipts, and screenshot checks before spending hosted CI minutes.

Cost discipline

Fast systems should not burn expensive context to prove routine work.

The same principle that makes Flywheel safe for SMB workflows makes it cheaper to operate: grep before fetching, batch routine checks, route the right model to the right task, and save hosted CI for final confirmation.
