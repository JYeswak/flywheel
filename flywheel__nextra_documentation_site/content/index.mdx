The agentic-coding flywheel: ZestStream's substrate for orchestrating coding
agents at scale, plus the operational doctrine that keeps the substrate
self-correcting.

This docs site is the canonical surface for flywheel's recipes, contracts,
and disciplines — sourced from `.flywheel/doctrine/` in the flywheel repo
and organized by Diátaxis quadrant.

## Audience personas

- **orch** — orchestrator pane (flywheel:1); reads doctrine + dispatch contracts
- **worker** — worker pane (flywheel:0.N); reads tick + callback contracts
- **Joshua** — operator/decision-maker; reads mission + paradigm anchors
- **operator** — external operator running flywheel against client repos
