# Fleet Coherence Schema v2

Status: frozen for Phase 0 fixtures on 2026-05-01.
Source: `/tmp/plan_fleet_coherence_synthesis_v2.md` lines 59-79 and 93-204.
Bead: `flywheel-2te`.

This document freezes the row contracts for fleet-coherence drift events,
suppression records, dedupe keys, and fixture data. It is a plan/spec artifact
only. Scanner, writer, validator, launchd, and remediation code are out of
scope for Phase 0.

## Drift States

Every event row carries these lifecycle fields:

- `state`: one of `open`, `still_open`, `closed`, `suppressed`
- `first_seen_ts`: first detector observation for the dedupe key
- `last_seen_ts`: newest detector observation for the dedupe key
- `seen_count`: number of detector observations rolled into this state
- `resend_after_ts`: earliest time an alert/report may re-fire

Close rows MUST use the same `dedupe_key` as the open row and set
`state="closed"`. Suppressed rows MUST name `suppression_id`.

## Event Row v2

Event rows are JSON objects with `record_type="event"` in fixture files. Runtime
event JSONL MAY omit `record_type` if the file stores events only.

Required fields:

| Field | Type | Notes |
|---|---|---|
| `ts` | RFC3339 string | Row emission time. |
| `schema_version` | integer | Must be `2`. |
| `detector` | string | Canonical value: `fleet-coherence`. |
| `detector_version` | string | Detector package/script version. |
| `detector_git_sha` | string | Build/source sha or `unknown`. |
| `event_id` | string | Stable unique event-row id. |
| `class` | enum | One drift class from the dedupe table. |
| `severity` | enum | `info`, `warn`, or `error`. |
| `state` | enum | `open`, `still_open`, `closed`, `suppressed`. |
| `session` | string | NTM session name. |
| `pane` | integer/null | Pane number when class is pane-scoped. |
| `dedupe_key` | string | Generated by class grammar below. |
| `first_seen_ts` | RFC3339 string | First observation for dedupe key. |
| `last_seen_ts` | RFC3339 string | Latest observation for dedupe key. |
| `seen_count` | integer | Must be >= 1. |
| `resend_after_ts` | RFC3339 string/null | Null only for closed rows. |
| `confidence` | number | 0.0 to 1.0 classifier confidence. |
| `classification_source` | string | Input families used for classification. |
| `sample_window_s` | integer | Time window sampled. |
| `sample_count` | integer | Count of raw samples considered. |
| `source_ts` | RFC3339 string/null | Timestamp of newest source record. |
| `source_age_s` | integer/null | Age at emit time. |
| `raw_source_refs` | array | Objects with `path` and optional `line`. |
| `evidence` | object | Class-specific observed facts. |
| `actions` | object | Shadow/remediation intent and receipts. |
| `l61` | object | Dual-channel comms audit fields. |
| `l62` | object | Skill-discovery callback audit fields. |
| `l63` | object | Recovery-drill audit fields. |
| `suppression_id` | string/null | Required when `state="suppressed"`. |

`actions` required keys:

- `shadow_mode`: boolean
- `would_l61`: boolean/null
- `would_bead`: boolean/null
- `would_no_bead_reason`: string/null
- `receipt_required`: boolean
- `bead_id`: string/null
- `no_bead_reason`: string/null

`l61` required keys:

- `project_key`
- `fleet_mail_identity_source`
- `vault_token_validated`
- `agent_mail_attempted`
- `agent_mail_sent_at`
- `agent_mail_message_id`
- `agent_mail_from`
- `agent_mail_to`
- `ntm_attempted`
- `ntm_sent_at`
- `ntm_session`
- `ntm_pane`
- `ntm_result`
- `l61_pairing_status`
- `degraded_reason`

Valid `l61_pairing_status` values are `not_attempted`, `complete`,
`mail_only`, `ntm_only`, `degraded`, and `suppressed`.

`l62` required keys:

- `repair_callback_required`
- `sd_count`
- `sd_ids`

`l63` required keys:

- `recovery_action_requires_drill`
- `recovery_drill_ids`

## Suppression Row v2

Suppression rows are JSON objects with `record_type="suppression"` in fixture
files. Required fields:

| Field | Type | Notes |
|---|---|---|
| `id` | string | Stable suppression id. |
| `created_at` | RFC3339 string | Creation time. |
| `created_by` | string | Human or agent identity. |
| `source_event_id` | string | Event that caused suppression. |
| `class` | string | Primary class being suppressed. |
| `session` | string | Required; no global class-only suppressions. |
| `pane` | integer/null | Required for pane-scoped classes unless pattern scopes it. |
| `dedupe_key_pattern` | string | Glob-style pattern, never just class. |
| `allowed_classes` | array | Explicit class allow-list. |
| `expires_at` | RFC3339 string | Mandatory expiry. |
| `max_ttl` | string | Human-readable cap, e.g. `2h`. |
| `review_due` | RFC3339 string | Review time before or at expiry. |
| `reason` | string | Operational reason. |
| `bead_id` | string/null | Required for high-severity unless no-bead reason exists. |
| `no_bead_reason` | string/null | Explicit alternative to `bead_id`. |

Suppression matching rules:

- A suppression MUST NOT match by class alone.
- A suppression MUST include `session` and either `pane` or a narrow
  `dedupe_key_pattern`.
- A high-severity suppression MUST include `bead_id` or `no_bead_reason`.
- Expired suppressions are ignored by detectors and retained for audit.

## Fixture Corpus Contract

Canonical fixture path:
`.flywheel/fixtures/fleet-coherence-fixtures.jsonl`

Fixture requirements:

- Valid JSONL, one object per line.
- At least five `record_type="event"` rows per drift class above.
- At least one closed-row example per class using the same grammar.
- Suppression rows use the same file with `record_type="suppression"`.
- ALPS-derived fixtures MUST include:
  - topology mismatch in `topology_stale_or_kind_mismatch`
  - Codex auth text in `codex_auth_expired_silent`
  - sustained pause receipt in `sustained_operator_pause_exceeded`
- Additional synthesis anchors represented in fixtures:
  - missing topology
  - invalid or missing token
  - CronCreate schedule drift
  - Codex false-idle / activity misclassification

Smoke validation:

```bash
jq -c . .flywheel/fixtures/fleet-coherence-fixtures.jsonl >/dev/null
jq -r 'select(.record_type=="event") | .class' .flywheel/fixtures/fleet-coherence-fixtures.jsonl | sort | uniq -c
```
